## 👨‍💻 About Me

I am a backend-focused Full-Stack Developer with a deep passion for building scalable, distributed systems and bringing the web to life through immersive 3D experiences. Whether routing complex data under the hood or rendering real-time graphics in the browser, I love bridging the gap between heavy logic and beautiful UI.

- 🔭 **Currently focusing on:** Building highly scalable systems & real-time applications.
- 🌍 **Passionate about:** 3D & Immersive Web technologies (Three.js / Babylon.js).
- 🔌 **Specialty:** Distributed systems, microservices, and seamless API integrations.
